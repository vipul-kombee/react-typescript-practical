### React Interview 1.0.0 Frontend Repo

## Requirements

- Node v20.11.1
- React v18.3.1
- TypeScript v5.x
- Vite v5.x
- Bootstrap v5.x
- npm v10.2.4

## Install Dependencies

```bash
git clone [clone-url]

git checkout [working-branch]

npm install

```

## Running the App

# Start the development server

npm run dev

# Build for production

npm run build
